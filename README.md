## Develop
```bash
$ git clone git@github.com:goldylucks/adamgoldman.me.git
$ cd adamgoldman.me
$ yarn install
$ yarn start # starts node server at port 3000
```
open your favorite browser and [hack away](http://localhost:3000/)!
